Compétences

Diplômée de la formation "Développeur web full-stack" chez Handigital en 2021, découvrez mes domaines de compétences et venez parcourir mes réalisations dans mon Portfolio.

Nous avons appris à intégrer des maquettes grâce aux langages HTML, CSS et JS. Nous avons notamment appris à utiliser le framework Boostrap.

Exemples de compétences acquises :

  • Utiliser l'inspecteur de Chrome pour modifier des styles CSS en direct
  • Traduire une maquette simple en HTML/CSS
  • Traduire une maquette en page en utilisant Bootstrap
  • Développer des pages en Responsive : media queries

Nous avons appris le langage Javascript.

Exemples de compétences acquises :

  • Utiliser des variables, des conditions et des fonctions
  • Comprendre le DOM
  • Manipuler des tableaux et des objets
  • Utiliser jQuery

Tout au long de la formation, nous avons travaillé sous deux environnements : Windows et Linux. J'ai installé une machine virtuelle pour pouvoir utiliser Linux Mint.

Exemples de compétences acquises :

  • Utiliser Linux : connaître les commandes
  • Gérer des fichiers Linux : identifier, modifier ou supprimer des droits
  • Administrer un système Linux : gérer les utilisateurs
  • Créer des Scripts Shell

Nous avons utilisé PHP pour réaliser des sites dynamiques.

Exemples de compétences acquises :

  • Accéder aux bases de données avec le langage PHP
  • Connaître la programmation orientée objet
  • Traiter des formulaires avec le langage PHP : utiliser les paramètres GET/POST pour récupérer les données
  • Mettre en oeuvre le modèle Model-View-Controller (MVC)

Nous avons appris à créer une base de données.

Exemples de compétences acquises :

  • Concevoir une base de données
  • Schématiser le modèle Entité - Association et UML : diagramme de classe
  • Se connecter à MySQL
  • Créer une base de données MySQL
  • Effectuer des requêtes

Nous avons créé une maquette d'application mobile grâce au logiciel Adobe XD.

Exemples de compétences acquises :

  • Concevoir un Userflow
  • Réaliser un Zoning
  • Concevoir une Maquette et un Prototype
  • Réaliser un Style Guide et exporter les composants (assets)

Nous avons découvert le CMS (Content Management System) Wordpress.

Exemples de compétences acquises :

  • Administrer Wordpress : installer et configurer Wordpress
  • Parcourir Wordpress en mode éditeur : créer et modifier des pages, installer des thèmes, etc.
  • Développer un thème graphique : créer un thème enfant

Nous avons mis en place notre veille numérique et découvert la gestion de projet agile.

Exemples de compétences acquises :

  • Apprendre à faire une veille numérique efficace et pérenne
  • Travailler en mode agile
  • Utiliser des outils de gestion de projet
